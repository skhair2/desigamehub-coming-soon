## Blog Post Template

```markdown
---
title: "[Main Keyword]: [Benefit/Angle]"
description: "[120-150 chars] Main keyword naturally included"
keywords: [primary, secondary, long-tail keywords]
author: "DesiGameHub Team"
publishedDate: "YYYY-MM-DD"
readTime: "X min read"
---

# [H1: Main Keyword + Power Word]

## Introduction (150-200 words)
- Hook with problem/question
- State what reader will learn
- Mention DesiGameHub's solution

## Table of Contents
- Link to all H2 sections

## [H2: Main Point 1]
### [H3: Sub-point]
- Content optimized for keyword
- Natural keyword usage (1-2%)
- Internal link opportunity

[Repeat for 3-5 main points]

## FAQ Section
**Q: Related question?**
A: Answer with keyword naturally included

## Conclusion (150-200 words)
- Summarize key points
- Restate main benefit
- Strong CTA to DesiGameHub platform

## CTA Box
"Ready to [Action]? [Link to feature/signup]"
```

## SEO Checklist for Each Blog Post

Before Publishing:
- [ ] Title contains primary keyword (50-60 characters)
- [ ] Meta description has primary keyword (150-160 characters)
- [ ] Internal links: 3-5 contextual links
- [ ] Outbound links: 1-2 to authoritative sources
- [ ] Images: All have descriptive alt tags
- [ ] Keyword density: 1-2% for primary keyword
- [ ] Headings: H1, 2-3 H2s, H3s under each H2
- [ ] Length: 1500-3000 words depending on topic
- [ ] Mobile optimized: Test on mobile view
- [ ] Schema markup: Add if applicable
- [ ] Internal link anchor text: Keyword-rich, natural
